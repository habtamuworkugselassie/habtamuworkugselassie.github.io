Real-Time GPS Tracking System

×
Full-Stack Java Netty PostgreSQL High-Concurrency

Overview

As the lead engineer at Gize Trading PLC, I architected and developed a comprehensive real-time GPS tracking system used to monitor over 2,000 motorbikes across Addis Ababa for the Addis Ababa City Road Authority.

Key Features

  • Real-time GPS position tracking
  • Support for multiple GPS device protocols
  • Trip analytics and reporting
  • Vehicle monitoring dashboards
  • High-concurrency handling (thousands of simultaneous connections)

Technical Implementation

Implemented a Netty-based UDP/TCP server for tracking device connections, Jetty web server, and Google Guice dependency injector to handle concurrent device connections and real-time updates. Used PostgreSQL for data persistence and optimized I/O handling in Netty for improved system reliability and performance.

Impact

The system successfully monitors over 2,000 vehicles in real-time, providing the Addis Ababa City Road Authority with comprehensive tracking and analytics capabilities. The high-concurrency architecture ensures stable performance even under heavy load.
